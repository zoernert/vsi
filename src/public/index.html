API Documentation

Explore the REST API with our comprehensive documentation:

OpenAPI Specification Swagger UI Guide

Key Endpoints:

  • GET /collections - List collections
  • POST /collections/{name}/points/search - Search vectors
  • PUT /collections/{name}/points - Upsert points
  • POST /api/collections/{name}/search - Semantic search
  • /api/v1/qdrant/* - Direct Qdrant proxy

Authentication:

Use JWT Bearer tokens or API keys in request headers:

Authorization: Bearer your-jwt-token
# or
api-key: your-api-key

Quick Example

Using the service with a Qdrant client:

// JavaScript
import { QdrantClient } from '@qdrant/qdrant-js';

const client = new QdrantClient({
  url: 'http://localhost:3000',
  apiKey: 'your-jwt-token'
});

const results = await client.search('my_collection', {
  vector: [0.1, 0.2, 0.3, ...],
  limit: 10
});
